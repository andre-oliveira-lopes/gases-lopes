Sobre o Sistema

O Gases União é uma aplicação desktop desenvolvida com Electron, HTML, CSS (SCSS) e JavaScript (Node.js), projetada para simplificar e otimizar o controle de pedidos, entregas e estoque de cilindros de gás.

O sistema oferece uma interface moderna e responsiva, com foco em desempenho, usabilidade e facilidade de manutenção. Cada módulo foi desenvolvido para atender às necessidades do fluxo operacional de empresas do setor de gases industriais e medicinais.

Informações do Projeto

  • Nome do aplicativo: Gases União
  • Versão: 1.0.0
  • Plataforma: Windows (Electron)
  • Banco de dados: SQLite
  • Linguagem principal: JavaScript (Node.js + Electron)
